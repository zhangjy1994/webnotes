# 键盘快捷方式

  • command+b（展开/折叠左侧菜单）
  • command+\（左右分屏）
  • command+1/2（切换分屏）
  • command+option+F（替换）
  • command+shift+F（全局查找）
  • command+option+ -> / <-（切换tab）
  • command+shift+p + reload（重新加载窗口）
