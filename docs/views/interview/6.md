2. 箭头函数和普通函数的区别
  * 箭头函数 和 普通函数的this指向的区别
  * 箭头函数没有arguments
  * 箭头函数不能new --- 因为没有this
  * 箭头函数不能bind call apply 改变this指向
  ``` js
  const fn = () => {
    console.log(this);
    console.log(this.name)
  }
  const test = {
    name: 'zangjiawei'
  }
  fn.call(test)
  ```
  * 没有原型属性
